# Linked Lists in Rust

A learning project implementing various linked list data structures in Rust.

This project follows the excellent guide: [Learn Rust With Entirely Too Many Linked Lists](https://rust-unofficial.github.io/too-many-lists)

## Purpose

This library explores different approaches to implementing linked lists in Rust, including:
- Singly linked lists
- Doubly linked lists
- Persistent data structures

By working through these implementations, we hope to learn about Rust's ownership system, borrowing rules, and patterns for managing complex data structures.

## Building

```bash
cargo build
```

## Running Tests

```bash
cargo test
```
